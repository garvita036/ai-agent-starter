# AI Agent Starter

Minimal, extendable Python AI Agent that demonstrates:
- planning with an LLM (OpenAI)  
- tool calling (calculator, file reader)  
- interactive CLI loop

## Features
- Small, modular codebase
- Easy to add tools
- Works in mock mode (no API key) for offline testing
